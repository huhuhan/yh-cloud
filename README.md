## 项目简介

### 技术框架

- Spring框架版本：Spring Cloud Hoxton.SR5 | Spring Boot 2.2.10.RELEASE
- 注册中心：Eureka
- 服务间通信：OpenFeign
- 服务熔断：Hystrix
- 统一网关：Spring Cloud Gateway
- 认证鉴权：Spring Cloud Oauth2 + JWT
- 服务监控：Spring Boot Admin



- 工作流引擎：Activiti



### 运行前提

配置Hosts
```python
127.0.0.1 yh-eureka
127.0.0.1 yh-gateway
```
